Introduction to HTML by OLEH

Doctype

Description

All HTML documents must start with adeclaration

The declaration is not an HTML tag. It is an "information" to the browser about what document type to expect.

Example

<!doctype html>

HTML

Description

The tag represents the root of an HTML document.

The container for all other HTML elements (except for the <!DOCTYPE> tag).

Example

<html>

</html>

Head

Description

The <head> element is a container for metadata (data about data) and is placed between the <html> tag and the <body> tag.

Metadata is data about the HTML document. Metadata is not displayed.

Metadata typically define the document title, character set, styles, scripts, and other meta information.

Example

<head>

</head>

Body

Description

The <body> tag defines the document's body.

The <body> element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

Example

<body>

</body>

Title

Description

The <title> tag defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab.

The <title> tag is required in HTML documents!

The contents of a page title is very important for search engine optimization (SEO)! The page title is used by search engine algorithms to decide the order when listing pages in search results.

Example

<title>

</title>

p

Description

The <p> tag defines a paragraph.

Browsers automatically add a single blank line before and after each <p> element.

Example

<p>Ths is a paragraph</p>

Works example

Ths is a paragraph

span

Description

The <span> tag is an inline container used to mark up a part of a text, or a part of a document.

The <span> tag is easily styled by CSS or manipulated with JavaScript using the class or id attribute.

The <span> tag is much like the <div> element, but <div> is a block-level element and <span> is an inline element.


Example

<p>This is a <span style="color:red">inline span.</span>This is outside the span</p>

Works example

This is a inline span.This is outside the span

div

Description

The <div> element is used as a container for other HTML elements.

The <div> element is by default a block element, meaning that it takes all available width, and comes with line breaks before and after.

The <div> element has no required attributes, but style, class and id are common.

The <div> element is often used to group sections of a web page together.

Example

<div style="background-color:yellow"><p>This is the div element.</p></div>

Works example

This is the div element.

Heading elements-h1 to h6

Description

HTML headings are titles or subtitles that you want to display on a webpage.

HTML headings are defined with the <h1> to <h6> tags.

<h1> defines the most important heading. <h6> defines the least important heading.

Example

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

formatting elements-strong, emphasis

Description

The HTML <strong> element defines text with strong importance. The content inside is typically displayed in bold.

The HTML <em> element defines emphasized text. The content inside is typically displayed in italic.

Example

<p>This text is <strong>IMPORTAND.</strong></p>

<p>This text is <em>EMPHASIZED.</em></p>

Works example

This text is IMPORTAND.

This text is EMPHASIZED.

tables & caption

Description

HTML tables allow web developers to arrange data into rows and columns.

A table in HTML consists of table cells inside rows and columns.

Each table cell is defined by a <td> and a </td> tag.

Everything between <td> and </td> is the content of a table cell.

Example

<td>15</td>
</tr>
<tr>

</tr>
</tbody>
</table>

lists

Description

HTML lists allow web developers to group a set of related items in lists.

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag.

An ordered list starts with the <ol> tag. Each list item starts with the <li> tag.

The <dl> tag defines the description list, the <dt> tag defines the term (name), and the <dd> tag describes each term

Example

<ul>
<li>Coffee</li>
<li>Milk</li>
<li>Tea</li>
</ul>
<ol>
<li>Coffee</li>
<li>Milk</li>
<li>Tea</li>
</ol>
<dl>
<dt>Coffee</dt>
<dd>black hot drink</dd>
<dt>Milk</dt>
<dd>cold white drink</dd>
</dl>

images & Alt

Description

Images can improve the design and the appearance of a web page.

The HTML <img> tag is used to embed an image in a web page.

The <img> tag creates a holding space for the referenced image.

The required alt attribute provides an alternate text for an image, if the user for some reason cannot view it (because of slow connection, an error in the src attribute, or if the user uses a screen reader).

Example

<img src="images/car_small.jpg" width="480" height="320" alt=""/>

Works example

links

Description

Links are found in nearly all web pages. Links allow users to click their way from page to page.

Example

<a href="https://www.discoveryeducation.com/">Discovery Education</a>

Works example

Discovery Education

quote & blockquote

Description

The HTML <blockquote> element defines a section that is quoted from another source.

Browsers usually indent <blockquote> elements.

The HTML <q> tag defines a short quotation.

Browsers normally insert quotation marks around the quotation.

Example

<p>&quot;I realy like an animals.&quot;</p>

<blockquote>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Et accusantium excepturi nobis. Mollitia voluptatem aut laboriosam labore quae quasi voluptates dignissimos vitae repudiandae, deserunt blanditiis. Ea soluta illum at fugiat.</blockquote>

Works example

"I realy like an animals."

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Et accusantium excepturi nobis. Mollitia voluptatem aut laboriosam labore quae quasi voluptates dignissimos vitae repudiandae, deserunt blanditiis. Ea soluta illum at fugiat.

abbreviations

Description

The HTML <abbr> tag defines an abbreviation or an acronym, like "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".

Marking abbreviations can give useful information to browsers, translation systems and search-engines.

Example

<p>The <abbr title="United Nations">UN</abbr> created on October 24,1945</p>

Works example

The UN created on October 24,1945

meta

Description

Metadata is used by browsers (how to display content or reload page), search engines (keywords), and other web services.

Example

<meta name="keywords" content="HTML,CSS,Code">

<meta name="description" content="This is my website">

Special symbols and character entities:

Description

Symbols or letters that are not present on your keyboard can be added to HTML using entities.

To add such symbols to an HTML page, you can use the entity name or the entity number (a decimal or a hexadecimal reference) for the symbol

Example

<p>I will display &euro;</p>
<p>I will display &copy;</p>
<p>I will display &reg;</p>
<p>I will display &trade;</p>
<p>I will display &larr;</p>
<p>I will display &spades;</p>
<p>I will display &amp;</p>
<p>I will display a&#769;</p>
